# NBA Trivia Showdown

An endless NBA trivia game featuring a live 1-on-1 physics-based basketball simulation underneath the answer box.

## Features

- **LED Retro Scoreboard**: Tracks HOME (User) vs AWAY (Opponent) score, high scores, accuracy statistics, and a running game clock.
- **Pre-Shot Possession Transition Phase**: Ball zips to the shooter's hands before a shot.
- **Directional Arc Shooting**: Correct answers shoot toward the right hoop; incorrect ones toward the left hoop.
- **On-Fire Ball Animation**: Ball catches fire and leaves a trailing smoke/fire aura when you hit 3 or more correct answers in a row.
- **21-Point Match**: Play a complete match to 21 points with single point increments.
- **Clean Court Layout**: Neon grid styling with dual mirrored hoops and nets.
